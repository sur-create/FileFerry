# 局域网文件传输工具 V1.3 需求文档

## 1. 版本目标

在 V1.2（多文件/目录传输）基础上，新增跨平台桌面 GUI，重点覆盖 Linux 与 macOS，提供中文界面与手动连接控制。

## 2. 功能需求

- FR-301：提供桌面 GUI 主窗口（中文界面）。
- FR-302：发送端支持手动“开启连接/断开连接”。
- FR-303：接收端支持手动“开启连接/断开连接（监听）”。
- FR-304：GUI 支持添加多文件/目录并调用 V1.2 传输能力。
- FR-305：GUI 支持冲突策略（覆盖/跳过/重命名）与失败继续策略。
- FR-306：GUI 显示会话摘要与条目错误日志。
- FR-307：Linux/macOS 安装包包含 GUI 可执行入口与桌面启动项。

## 3. 非功能需求

- NFR-301：界面文字为中文。
- NFR-302：传输任务在后台线程运行，界面不可卡死。
- NFR-303：保留 CLI 能力，GUI 与 CLI 可并存。

## 4. 验收标准

1. Linux/macOS 启动 GUI 正常，中文文案完整。
2. 发送端可手动连接后发送，断开后不可发送。
3. 接收端可手动开启监听与关闭监听。
4. GUI 发起多源传输，核心行为与 V1.2 一致。
5. CLI 回归测试全部通过。
